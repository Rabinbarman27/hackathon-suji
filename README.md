### Key Concepts

#### 1. Authentication (Mock)
We use a simple "mock" authentication system. Instead of talking to a server, we save the user's credentials in `localStorage`.
*See `script.js` for implementation details.*

#### 2. Data Persistence
The app remembers your progress using the browser's `localStorage` API.
- `isLoggedIn`: Status flag.
- `points`: Earned score.
- `studyProgress`: JSON object identifying study minutes by date.

#### 3. Navigation
Multipage navigation is handled via standard `window.location.href` redirects.
